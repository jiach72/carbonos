## v1.0 阶段：全模拟数据驾驶舱 🎮
核心目标：在无真实业务接入的情况下，通过高质量的模拟数据展示系统的全部能力。

### 模块 A：IoT 仿真引擎 (优先级：最高)
让 Dashboard "活"起来，展示逼真的实时跳动数据。
- [ ] 多场景模拟：工厂、数据中心、商业园区
- [ ] 数据生成器：模拟电力/水/气/光伏的 24h 实时曲线
- [ ] 告警模拟：随机生成设备故障与恢复事件

### 模块 B：产品碳足迹 (PCF) 演示
- [ ] 预置案例：锂电池、服务器机柜的碳足迹计算演示
- [ ] 报告生成：一键生成模拟的 ISO 14067 报告

### 模块 C：AI 预测 (基于模拟数据)
- [ ] 趋势预测：基于生成的一年历史数据，预测下月能耗
